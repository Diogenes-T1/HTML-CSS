Imagens Dinâmicas

Esse tipo de comando serve para o uso da mesma imagem em diversos dispositivos.

Para isso usa-se a tag "picture" e dentro a tag "img"

Importante usar a tag "source" com o comando "media type" para dinamizar a imagem na própria página

Ficando assim:

>

(Diminua o tamanho da página para ver a imagem diminuindo)

Meus Áudios

Para inserir um áudio, usa-se a tag "audio" com o comando src= junto com "controls autoplay"

ficando assim:

Battleground - Ethan Meixsell.mp3

Amazer Lazer - Audio Hertz.mp3

Acoustic Blues - Audionautix.mp3 (Acoustic Blues de Audionautix é licenciada de acordo com a licença Atribuição 4.0 da Creative Commons. https://creativecommons.org/licenses/by/4.0/ Artista: http://audionautix.com)

VÍDEOS HOSPEDADOS LOCALMENTE

Para inserir vídeos hospedados localmente usa-se a tag "videos" e o comando "width=" para escolher o tamanho

Assim como nos audios, usar também o comando "controls" para aparecer os botões de controle

e caso queira usar uma capa no vídeo usar o comando "poster=" dentro da tag video

Ficando assim:

Macaco simpático comendo maçã
