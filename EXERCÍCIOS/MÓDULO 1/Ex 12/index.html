VÍDEOS HOSPEDADOS LOCALMENTE

Para inserir vídeos hospedados localmente usa-se a tag "videos" e o comando "width=" para escolher o tamanho

Assim como nos audios, usar também o comando "controls" para aparecer os botões de controle

e caso queira usar uma capa no vídeo usar o comando "poster=" dentro da tag video
