PRINCIPAIS FORMATAÇÕES

NEGRITO / DESTAQUE

Nesta frase temos um Termo em Negrito usando a tag B (não semântico)

Nesta frase temos um Termo em destaque usando a tag STRONG (semântico)

ITÁLICO / ÊNFASE

Nesta frase temos um termo em Itálico usando a tag i (não semântico)

Nesta frase temos um termo em Ênfase usando a tag EM (semântico)

TEXTO MARCADO

Nesta frase temos uma marcação com a tag MARK (não semântico, portanto para mudar a cor usa-se a linguagem css)

TEXTO DELETADO

Pode-se marcar um texto como deletado para indicar que ele deve ser lido, mas não considerado

TEXTO INSERIDO

Já o inserido pode ficar marcado com uma ênfase e que foi adicionado depois.

Existe também a tag U para indicar sublinhação porém (não semântica)

TEXTO SOBRESCRITO

Pode-se escrever expressões do tipo x10+2

TEXTO SUBESCRITO

Pode-se usar para escrever expressões do tipo H2O
