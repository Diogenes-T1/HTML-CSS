Listas

Ordered Lists

Para criar listas usa-se a tag "ol" e para delimitar, usa-se a tag "li"

Como por exemplo a seguinte lista:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5

Unordered Lists

É a mesma coisa, porém a lista fica sem ordenação, para isso usa-se a tag "ul" e a tag "li" da mesma forma.

Ficando assim:

  • Item 1
  • Item 2
  • Item 3
  • Item 4

OBS: dá para alterar o tipo de marcação usando os valores disc, circle e square dentro das tags.

Listas Misturadas

É possível mesclar entre os tipos de listas alternando as tags, como por exemplo:

  1. Item 1
  2. Item 2
    1. Item 1
    2. Item 2
    3. Item 3
  3. Item 3
  4. Item 4
    • Item A
    • Item B

Lista de definição

Como o proprio nome diz é uma lista para descrever os componentes. Para isso Usa-se as Tags "dt" para o componente da lista e "dd" para a descrição.

Como por exemplo:

HTML
Linguagem de marcação para criação de conteúdo de um site.
CSS
Linguagem de marcação para a criação de design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.
