Usando links externos

Para indexar links usa-se a tag "a" com a referencia em css "href="

como por exemplo:

Acesso ao meu Facebook

Porém é muito melhor que o link abra em uma aba nova, para isso usa-se os comandos: "target=_blank" (aba nova) e rel="external" (link externo)

Como por exemplo

Links para Download

para fazer links para download, pode-se criar um lista e dentro a tag "a" normalmente

Ficando assim:

  • Download Fac's IBGE

      • É importante identificar para os algorítmos que vc tem um download e o tipo usando as referências em Css: "download=" e "type="

  • Download Facs IBGE zipado
